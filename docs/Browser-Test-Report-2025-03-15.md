## 2. Найденные баги и замечания UI

### 2.1 Критичные

| № | Описание | Путь воспроизведения |
|---|----------|------------------------|
| 1 | **404 на /portal при первом редиректе после логина студента** | Войти как student1@test.local → редирект на /portal → кратковременно 404. При повторной загрузке или переходе на /portal/student/dashboard страница отображается. Возможно, race condition при SSR/редиректе. |

### 2.2 Замечания

| № | Описание | Путь воспроизведения |
|---|----------|------------------------|
| 1 | **SCORM-плеер: курсы без контента** | Курс course-seed-31 (и др.) показывает «У этого курса пока нет загруженного SCORM-контента». В seed только первые 5 published курсов получают scormPath, но физические файлы могут отсутствовать в `public/uploads/`. |
| 2 | **Клик по карточке курса** | На странице «Мои курсы» клик по названию курса иногда не переводит на страницу курса — требуется прямая навигация. |
| 3 | **Редирект после signOut** | При выходе через /signout редирект может вести на localhost:3000 вместо 3001 (если NEXTAUTH_URL или callbackUrl указывает на другой порт). |

## 3. Ошибки и предупреждения консоли

| Тип | Сообщение | URL/контекст |
|-----|-----------|--------------|
| **debug** | `[next-auth][error][CLIENT_FETCH_ERROR] Failed to fetch` | /signout — при выходе |
| **warning** | `[CursorBrowser] Native dialog overrides installed` | Системное (MCP) |
| **warning** | `Download the React DevTools for a better development experience` | Стандартное React |
| **debug** | `Warning: Expected server HTML to contain a matching <header> in <body>` | localhost:3000 — гидратация |
| **debug** | `Uncaught Error: Hydration failed because the initial UI does not match what was rendered on the server` | localhost:3000 |
| **debug** | `Warning: An error occurred during hydration. The server HTML was replaced with client content` | localhost:3000 |
| **debug** | `Uncaught Error: There was an error while hydrating. Because the error happened outside of a Suspense boundary, the entire root will switch to client rendering` | localhost:3000 |
| **debug** | `Warning: Extra attributes from the server: data-cursor-ref` | Footer.tsx (login) — атрибут от Cursor IDE для автоматизации |

**Примечание:** Ошибки на localhost:3000 могли возникнуть при редиректе signOut на другой порт. На localhost:3001 основные страницы работают без критичных ошибок.

## 4. План доработки (приоритизированный)

### Высокий приоритет

1. **Исправить 404 на /portal при первом редиректе** — разобраться с race condition при редиректе студента с /portal на /portal/student/dashboard.
2. **Проверить callbackUrl при signOut** — убедиться, что редирект после выхода ведёт на тот же хост/порт, с которого зашли.

### Средний приоритет

3. **SCORM-контент для тестовых курсов** — добавить минимальный тестовый SCORM-пакет в seed или в `public/uploads/` для курсов с scormPath.
4. **Навигация по карточкам курсов** — проверить, что клик по карточке/названию курса стабильно переводит на страницу курса.

### Низкий приоритет

5. **Убрать/игнорировать data-cursor-ref в production** — атрибут добавляется Cursor IDE; в production его не будет. При необходимости — условный рендер.
6. **Hydration-ошибки** — проверить компоненты Header, Footer на несоответствие server/client HTML (например, условный рендер по `typeof window`).

---

## 5. Итог

Тестирование проведено по сценариям: гость, студент, менеджер, админ. Основной функционал работает. Выявлены: возможный 404 при первом редиректе в портал, замечания по SCORM и навигации. Консольные ошибки в основном связаны с гидратацией и Cursor IDE. Рекомендуется устранить пункты высокого приоритета перед релизом.
