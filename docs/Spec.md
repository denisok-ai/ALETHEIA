---

## 2. Функциональные блоки

### 2.1 Публичный сайт

- **Hero** — фото Татьяны, название, CTA
- **Почему AVATERRA** — ценности, цифры
- **Форматы** — 4 карточки (консультация, тренинг, курс, онлайн)
- **О мастере** — фото, история
- **Отзывы** — 3 отзыва
- **Записаться** — форма (имя, телефон, email, сообщение)
- **Футер** — логотип, контакты, оферта, политика

### 2.2 Авторизация

- Регистрация, вход, сброс пароля
- NextAuth (Credentials), сессия в cookies
- Роли: user, manager, admin

### 2.3 ЛК студента

- Дашборд (курсы, уведомления, шкала Энергии, бейджи)
- Мои курсы, SCORM-плеер
- Сертификаты (PDF)
- Медиатека, уведомления, профиль
- Поддержка (создание тикета)

### 2.4 Админка

- Пользователи (роли, статус)
- Курсы (создание, загрузка SCORM)
- CRM (лиды, конвертация, воронка)
- Оплаты (таблица, экспорт CSV)
- Медиа (загрузка)
- AI-настройки (чат-бот)
- Коммуникации, аудит, сертификаты

### 2.5 Кабинет менеджера

- Тикеты, поиск пользователей
- Верификация Phygital-заданий

---

## 3. Интеграции

| Сервис | Назначение |
|--------|------------|
| PayKeeper | Платежи, webhook → enrollment |
| Prisma + SQLite | БД (локально). Хранилище — public/uploads/ |
| Resend | Уведомления о заявках |
| Telegram | Бот, команды /start, /progress, /cert |
| DeepSeek | Чат-бот (настраивается через llm_settings) |

---

## 4. Референсы дизайна

- **Прототип:** Gamma (структура и контент)
- **Цвета:** primary #2D1B4E, secondary #D4AF37, dark #0A0E27
- **Шрифты:** Literata (заголовки), Outfit (текст)
- **Медиа:** docs/Media.md, docs/ImagePrompts.md

## 5. Ограничения

- Один язык (русский)
- Тексты секций — в коде (components/sections)
- Курсы — в БД, редактируются в админке
