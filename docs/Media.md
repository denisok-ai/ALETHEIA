## 2. Остальные изображения — в едином стиле

Остальные картинки (фон, декоративные блоки, иконки партнёров, обложки секций и т.д.) **не обязательно копировать с прототипа**: их можно **переделать или подобрать в том же визуальном стиле**.

**Рекомендуемый стиль (по прототипу Gamma):**
- **Настроение:** спокойный, доверительный, «здоровье и гармония» — без агрессивных контрастов и пёстрых цветов.
- **Цвета:** приглушённые, натуральные оттенки (беж, мягкий зелёный, тёплый белый, серый, мягкий синий); избегать кислотных и неоновых цветов.
- **Композиция:** простые, «дышащие» блоки; при использовании фото людей — естественные позы, доброжелательность.
- **Форматы:** для фонов и крупных блоков — растровые изображения (JPG/WebP); для иконок и логотипов — SVG или PNG с прозрачностью.
- **Разрешение:** для ретина-экранов готовить @2x там, где нужна чёткость (логотипы, ключевые иллюстрации).

**Где что может понадобиться:**
- Hero-секция: фоновое изображение или абстракция (природа, силуэты, текстура).
- История (4 шага): небольшие иконки или иллюстрации к каждому шагу.
- Услуги: опционально — иконка или мягкое фото к каждой карточке.
- Партнёры: логотипы Института им. Энгельгардта и РНИМУ им. Пирогова (официальные логотипы с сайтов организаций или из прототипа, с соблюдением правил использования).
- Отзывы: при желании — нейтральные аватары (или без фото).
- Футер: при необходимости — логотип ALETHEIA, мелкая графика.

## 3. Структура папок под медиа (рекомендация)

```
public/
  images/
    tatiana/          # Фото Татьяны Стрельцовой с прототипа
    partners/         # Логотипы партнёров
    sections/         # Фоны и иллюстрации секций (hero, история и т.д.)
    icons/            # Иконки (если не используем шрифт-иконки)
```

Либо аналог в `src/assets/` при использовании сборки с импортом статики.
