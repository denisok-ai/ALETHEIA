# Тематические изображения (Hero / About / Program)

Скопируйте сюда изображения в мистическом стиле:

- **hero-banner.png** — баннер: руки, ключ, светящиеся частицы, тёмно-синий/бирюзовый/золотой фон (~800–1200 px).

В папке **hero** (`public/images/hero/`):
- **hero-bg.png** — фон первого экрана: кристаллы/алмазы, золотой свет, фиолетово-синий космос. Эта картинка заполняет весь первый экран и хорошо видна.
- **about-path.png** — образ «пути», гармонии, духовной практики (~600–800 px).
- **program-energy.png** — энергия, свет, частицы для блока программы (~600–800 px).

Формат: PNG, JPG или WebP. После добавления файлов они автоматически отобразятся на сайте.

**Полный набор промптов для генерации** (Hero background, иконки модулей, автор, тарифы, декор, success, FAQ) — см. **docs/ImagePrompts.md**.
